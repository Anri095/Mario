# Mario pyramid
# Description
This project is a simple console-based Mario-style pyramid builder written in C. It was developed as part of CS50 coursework and helps demonstrate basic programming concepts such as loops, conditionals, and user input validation.
# What i learned
Using for and do-while loops
Handling user input in C
Input validation (restricting values between 1 and 8)
Building structured console output
Improving problem-solving skills with basic algorithms
